MASSAGE RESERVATION SYSTEM - PHP & MYSQL

Requirements:
1. XAMPP, Laragon, or any PHP/MySQL local server
2. PHP 8+ recommended
3. MySQL/MariaDB

Installation:
1. Copy the folder "massage_reservation_system" to your server directory:
   - XAMPP: htdocs
   - Laragon: www
2. Open phpMyAdmin.
3. Import database.sql.
4. Open config.php and update database username/password if needed.
5. Run the system in your browser:
   http://localhost/massage_reservation_system

Default Admin Account:
Email: admin@example.com
Password: admin123

Features:
- Customer registration/login
- View massage services
- Book appointment
- View customer reservations
- Admin dashboard
- Manage reservations and statuses
- Add/delete services
- Add/delete therapists

Note:
This is an educational starter system. For production, add CSRF protection, stricter validation, and deployment security settings.
